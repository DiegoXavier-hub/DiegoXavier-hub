## Selected Projects

- **Predictive Modeling for Undergraduate Research**  
  End-to-end modeling pipeline with CatBoost, LightGBM, and Logistic Regression, including Optuna tuning, threshold calibration, SHAP analysis, and robustness checks.

- **Business Survival Risk Radar**  
  Survival analysis for Brazilian companies using Kaplan-Meier and Cox models, combined with 12-month closure-risk modeling and clear analytical storytelling.

- **ML From Scratch**  
  Core machine learning algorithms rebuilt in Python and NumPy to deepen understanding of model behavior and mathematical foundations.

- **Full-Stack Portfolio Platform**  
  Next.js + Express + Firebase project with admin authentication, Firestore content management, bilingual resume handling, and deployment-ready workflows.

- **Dos Santos Plumbing Services**  
  Production website built with React and Vite, optimized for image performance, navigation clarity, and maintainable delivery.
