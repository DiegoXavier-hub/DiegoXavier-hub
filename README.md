## Featured Project

My recent TCC project is centered on predictive modeling of academic dropout in higher education, combining structured data preparation, descriptive analytics, machine learning experiments, model evaluation, and article writing in LaTeX.

Main topics explored in the project:

- Data extraction from academic history records
- Data cleaning and analytical base construction
- Exploratory data analysis and descriptive statistics
- Logistic Regression, Random Forest, XGBoost, LightGBM, CatBoost, and baseline models
- ROC, PR, calibration, confusion matrix, and threshold analysis
- SHAP-based model interpretation
- Technical reports, QA, and reproducibility checks
